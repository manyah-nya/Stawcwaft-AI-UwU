# Settings:
*map_dir*: ***DON'T CHANGE THAT***

*map_selection*: _**["Id" / "Random"]**_

*map_id*: _**[0 -> map_amount - 1]**_ Choose specific map from folder \n

*reports*: _**[True / False]**_ If True AI will give you game Reports

*debugs*: _**[True / False]**_ If True AI will give you data about it's learning process

*realtime*: _**[True / False]**_ Game in realtime

*enemy_race* _**["Random", "Terran", "Zerg", "Protoss"]**_ using "NoRace" is not recommended

*enemy_difficulty*:
- _**["VeryEasy", "Easy", "Medium", "MediumHard"]**_
- _**["Hard", "Harder", "VeryHard" "CheatVision", "CheatMoney", "CheatInsane"]**_
